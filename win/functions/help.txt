以下列出了支持的指令及其用法表述

其中，用 <> 括起来的参数为必填；
用 [] 括起来的参数为选填；
前后使用 | 号连接的代表必须选择其一填入。

---

    get <信息类型> <参数...>       获取指定类型的信息所对应的 API 接口返回值
    help                           查看程序帮助信息
    exit                           直接退出程序
    repo                           使用默认浏览器打开本程序的 GitHub 项目页面
    wiki                           使用默认浏览器打开本程序的官方 Wiki 页面

---

如需了解各个指令的详细用法，请访问 https://book.sotap.org 的相关条目。
如在使用中出现任何问题或有好的建议，可访问 https://github.com/sotapmc/commandline-reader/issues 填写反馈。
